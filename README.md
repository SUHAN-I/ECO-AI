## 🚀 Executive Summary

**ECO AI** is a Multimodal Generative AI platform purpose-built to digitize Pakistan's informal waste sector — and scale globally. By fusing **Llama 4 Scout Vision** for instant waste identification with **Llama 3.3-70B** for deep economic reasoning over a **FAISS-powered RAG pipeline**, ECO AI transforms discarded materials into traceable, profitable resources.

| What | Impact |
|------|--------|
| 📸 Photo → instant waste classification | Any smartphone, any condition |
| 💰 Real-time Misri Shah scrap pricing | Collectors bypass middlemen entirely |
| 🏭 Urdu-first manufacturing SOPs | Waste → industrial-grade products |
| 🚀 AI-generated micro-startup plans | Collectors become entrepreneurs |
| 🛂 Digital Product Passport | Corporate EPR compliance, 2026-ready |
| 🎤 Whisper voice interface | Solves literacy barrier at grassroots level |

Pakistan generates **30+ million metric tons** of municipal solid waste annually — yet only a fraction is formally recycled. ECO AI closes this gap by delivering industrial-grade intelligence directly to grassroots collectors, turning waste into wealth at scale.

## 💡 Our Solution

> ECO AI moves from a **Linear "Take-Make-Dispose"** model to a **Circular Economy**, creating "Green Startups" at the grassroots level.

| 🔧 Solution | 📝 How It Works |
|------------|----------------|
| **📸 Instant Material ID** | Snap a photo → Llama 4 Vision classifies waste (plastic, metal, glass, e-waste) in real Pakistani conditions |
| **💰 Real-Time Market Pricing** | RAG-powered Llama 3.3-70B fetches hyper-local Misri Shah rates — fair prices, not middleman rates |
| **🏭 Manufacturing Roadmap** | Step-by-step Urdu-first cleaning protocols + waste-to-product conversion guides |
| **🚀 Micro-Startup Toolkit** | AI-generated business plans: machinery costs, chemical procedures, buyer connections |
| **🛂 Digital Product Passport** | GPS-verified traceability for EPR compliance and material origin verification |
| **🎤 Voice Assistant** | Whisper AI STT — solves the literacy barrier for grassroots collectors |

## ⚡ Getting Started

### Prerequisites
```bash
Python 3.11+   Git
```

### Installation
```bash
# 1. Clone the repository
git clone https://github.com/YOUR-USERNAME/eco-ai.git
cd eco-ai

# 2. Install dependencies
pip install -r requirements.txt

# 3. Configure environment
cp .env.example .env
# → Edit .env with your API keys (see below)

# 4. Run locally
streamlit run app.py
```

### Required Secrets (`.env`)
```env
# ── AI (required) ───────────────────────────────────────────────────
GROQ_API_KEY=your_groq_api_key              # console.groq.com — free tier

# ── Image Storage (required) ────────────────────────────────────────
CLOUDINARY_CLOUD_NAME=your_cloud_name       # cloudinary.com — free tier
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret

# ── Vector Search (optional — falls back to in-memory) ──────────────
QDRANT_URL=https://your-cluster.qdrant.io   # cloud.qdrant.io — free tier
QDRANT_API_KEY=your_qdrant_api_key

# ── Logging (optional) ──────────────────────────────────────────────
GOOGLE_SHEETS_ID=your_spreadsheet_id
GOOGLE_SHEETS_CREDS_JSON=your_service_account_json

# ── Maps (optional) ─────────────────────────────────────────────────
GOOGLE_MAPS_API_KEY=your_maps_api_key
```

### Deploy to HuggingFace Spaces
```bash
git remote add hf https://huggingface.co/spaces/YOUR-USERNAME/eco-ai
git push hf main
```
Add secrets above in **HuggingFace Space → Settings → Repository Secrets**.
